# Prompt: Teach Me How Justfile Works And How To Use Justx

Copy this into your agent when you want a lesson before installing tools or changing a project.

```text
Please teach me how justfile works and how to use justx.

Use this tutorial if it is available:
tutorial/justfile-and-justx-tutorial.md

Please explain it like I am new to command runners.

Cover:
- what just is
- what a justfile is
- what recipes are
- how `just --list` works
- why agents benefit from a justfile
- how justx lets users browse and run recipes interactively
- what a safe beginner starter justfile looks like
- what commands I should try first

Do not edit any files during the lesson.

After the explanation, give me a small practice exercise and ask whether I want help installing just/justx or adding a justfile to a project.
```
